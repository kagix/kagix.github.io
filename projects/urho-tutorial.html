Urho 3D Tutorial

Urho 3D is an open-source game engine. If you cannot afford to pay for Unity or Unreal, this may be the answer for you. There is a catch. Open source software is often lacking good documentation. While Urho comes with a large number of examples, they are often too complex for the newcomer.

This is where this tutorial can help. It is based on the aforementioned examples, only at a slower pace. The tutorials are progressive, from the simplest to more complex ones.

Urho is Written in C++. Therefore, the tutorials use the same language. The information presented can easily be adapted to other programming languages, like C#. You can download the code of the tutorial for free from the repository.
