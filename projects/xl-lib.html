XL Lib

Excel is probably the most successful software program made by Microsoft. It is easy to use either as a GUI or from VBA. Like any Microsoft product, it starts to show its limits when processing big data. XL Lib has been created in order to use Excel with large sets of data. It can scale up to tens of thousands of worksheets and billions of rows.

It works on any modern OS, including Windows, OS X, Linux and the cloud.
